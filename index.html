¿Qué es HTML5?

HTML5 es la quinta versión del Lenguaje de Marcado de Hipertexto (HyperText Markup Language), el estándar utilizado para estructurar el contenido de las páginas web.

  • Próposito:

    • HTML5 organiza el contenido de una página en elementos como encabezados, párrafos, imágenes, tablas, listas, y multimedia (audio y video).

  • Características clave:
    • Soporte nativo para contenido multimedia, como "audio" y "video".
    • Introducción de nuevas etiquetas semánticas como "article", "section", y "header" que mejoran la accesibilidad y la optimización en motores de búsqueda (SEO).
    • Compatible con tecnologías modernas como aplicaciones web progresivas (PWAs).

¿Qué es CSS3?

CSS3 (Cascading Style Sheets, nivel 3) es el lenguaje de diseño que se utiliza para definir el aspecto visual de las páginas web.

  • Próposito:

    • CSS3 estiliza los elementos definidos en HTML5, controlando colores, fuentes, tamaños, posiciones y animaciones, entre otros aspectos visuales.

  • Características clave:
    • Nuevas propiedades como gradientes, sombras, y bordes redondeados.
    • Animaciones y transiciones sin necesidad de JavaScript.
    • Sistema de diseño avanzado con flexbox y grid.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado, esencial para agregar funcionalidad dinámica e interactividad a las páginas web.

  • Próposito:

    • Permite que las páginas web respondan a las acciones de los usuarios, como hacer clic en botones, rellenar formularios o cargar contenido dinámico sin recargar la página.

  • Características clave:
    • Manipulación del DOM (Document Object Model), que permite modificar el contenido y el diseño en tiempo real.
    • Animaciones y transiciones sin necesidad de JavaScript.Compatibilidad con bibliotecas y frameworks como React, Vue, y Angular.
    • Uso tanto en el cliente (navegador) como en el servidor (Node.js).
